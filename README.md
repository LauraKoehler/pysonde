# pySonde

## Setup
```sh
# Install dependencies
pipenv install --dev

# Setup pre-commit and pre-push hooks
pipenv run pre-commit install -t pre-commit
pipenv run pre-commit install -t pre-push
```

```
python setup.py sdist
pip install dist/pysonde*.tar.gz
```
